# Archive assets

This directory stores local artifacts that are useful for continuity but are not part of the
current automated training or evaluation entrypoints.

- `omni-audit/`: historical OMNI audit outputs copied from the service repo
- `runpod/`: local orchestration logs and dry-run traces
- `legacy-omni-audit/`: old scheduler and git-hook assets that referenced the previous Node-based audit flow

Treat these as reference material, not active pipeline code.
