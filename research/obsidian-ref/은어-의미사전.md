# 업계 은어 의미 사전

> **주의**: 모든 의미는 크롤 문맥에서 유추한 것. 일부는 추정이며 확신 수준 표시함.
> RAG·writer 프롬프트에서 이 단어를 쓸 때 **의미에 맞는 맥락**에서만 사용해야 자연스러움.

## 업소 타입 (확신: 높음)

| 은어 | 의미 | 문맥 예시 |
|---|---|---|
| **하퍼** | 하이퍼블릭(Hyper Public) — 룸살롱 상위 급수 | "강남 하퍼", "하퍼 초이스 기준이 몸매" |
| **쩜오** | 1.5차 업소 (중간 급수) | "쩜오 아가씨", "하퍼 쩜오", "레이블(쩜오) 티씨" |
| **텐카** | 텐프로+카라오케 (상위 급수) | "텐카 정쩜 언니들", "하퍼랑 스탈 다르죠" |
| **텐프로** | 텐카의 본딧말. 최상위 룸 | 거의 사용 안 됨, '텐'으로 축약 |
| **퍼블릭** | 공용 홀 업소 | "비강남 퍼블릭 상띠" |
| **쩜** | 쩜오 축약 | "쩜 점프", "정쩜" |
| **정쩜** | 정통 쩜오 | "정쩜 가봤는데 잘 안돼서" |
| **라운지** | 라운지 바 | "남친이랑 갈 라운지 추천" |
| **브압** | 브라질(특정 업소 타입? 추정) | "브압하고있는데", "브압 짭브" (확신: 중 — 특정 서비스 형태) |
| **ㅇㅍㄹ** | 오피스텔(오피) 초성 표기 | "ㅇㅍㄹ 와보세요 거기가 더함" |

## 업소 급수 (확신: 높음)

| 은어 | 의미 |
|---|---|
| **상띠** | 상위 티어 (외모·사이즈 기준 상급) |
| **중띠** | 중위 티어 |
| **하띠** | 하위 티어 |
| **중상띠/중하띠** | 중간+상/하 세분 |

*사용 맥락*: "강남ㅎㅍ 중띠정도에요", "텐 하띠 일퍼 중하띠"

## 감정·감탄사

| 은어 | 의미 |
|---|---|
| **샤갈** | 감정 한탄 — 슬픔·짜증의 구어 감탄 | "샤갈~~~ 진짜 이게 어떻게 된 일이냐고" |
| **ㅇㅍㄹ/ㅇㅅㅋ/ㅂㅇㅈ** | 특정 가게 초성 (의미 부정확) |
| **얼탱** | 어이없음 | "얼탱없어서" |

## 모름/불확실

- **ㅇㅅㅋ**: 1회 출현만 — 업소명 추정
- **ㄴㄷ**: 논현동? (지역 초성)
- **ㄱㄷㅂ**: 강동벌? (업소)
- **사라/도파/썸데이**: 특정 업소명
- **퀸**: 업계 플랫폼? (전반적으로 커뮤니티 맥락)
- **박카스**: 박카스 X — 특정 업태 또는 일반 명사

## Writer·RAG 활용 규칙

1. **의미 맞는 주제만**: "꼬빨/핸플"은 서비스 유형 상담 글에만. 일반 넋두리에 넣으면 어색.
2. **지역+업소 매칭**: "강남 하퍼", "서면 쩜오" 처럼 지역과 업소 묶어서.
3. **급수는 단문 질문에**: "상띠에요? 중띠에요?" 같은 짧은 패턴.
4. **업계 용어 밀도**: 크롤에서 post당 0.3-0.5개 업계어 포함. 과밀하면 부자연.
5. **초성 토큰**: ㅇㅍㄹ 등은 10자 미만 초단문에만 자연. 장문에 섞으면 어색.
