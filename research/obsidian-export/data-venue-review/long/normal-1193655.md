# ♾️ 금강산도 식후경 385 ♾️ 두쫀쿠

오늘 가게 맞은편 파이어랩 베이커리에서 두쫀쿠와 베이글 이벤트를 오픈 기념으로 했습니다

베이글 이벤트는 며칠 더하고 두쫀쿠도 재료 소진까지 하는 것 같습니답

1. 오후 1시 ~ 재료 소진까지 두쫀쿠 판매( 인당 두개 )

2. 오전 9시, 오전 11시 두번 시행, 음료 1잔 주문 시 선착순 50개 화덕 베이글 무료

3. 저는 앙버터 소금빵을 먹어봤는데 맛있습니다. ( 겉바속촉단짠 )

그래서 저는 밀크티(제로슈거)에 앙버터 소금빵 먹었는데
티랑 빵 둘다 맛있게 먹었습니답

두쫀쿠는 오후 3시 까지 있는 아기새들이 오늘 딱 두명 남아서
같이 마감 찍고 목 안 막히게 생과일쥬스랑 같이 줬어욥,

근데 이번 주가 징짜 춥긴하니까 다들 따뜻하게 입으십시오...
