## Dataview 예제 쿼리

### 전체 목록 (조회수 순)
```dataview
TABLE topic_label AS "주제", length_bucket AS "길이", views AS "조회", comment_count AS "댓글"
FROM "."
WHERE topic_cluster
SORT views DESC
LIMIT 50
```

### 바이럴 게시글만
```dataview
TABLE topic_label AS "주제", views AS "조회", slang_tokens AS "슬랭"
FROM "."
WHERE is_viral = true
SORT views DESC
```

### 클러스터별 질문 게시글
```dataview
TABLE topic_label AS "주제", length_chars AS "글자수"
FROM "."
WHERE has_question = true
GROUP BY topic_cluster
SORT length_chars DESC
```

### 긴 게시글 (deep reading)
```dataview
TABLE topic_label AS "주제", length_chars AS "글자수", honorific AS "말투"
FROM "."
WHERE length_bucket = "long"
SORT length_chars DESC
LIMIT 20
```
