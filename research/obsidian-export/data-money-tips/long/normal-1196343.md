# 그냥 대전 강남빌라에서 살고싶다

예전 직감이랑 촉 가지고 …
저 그냥 하던 ㅇㅍ 뛰어야 할까요 …
남자친구가 없으면 검은색 쥐들이 집으로 막 달려들어옴
그래서 남자친구 집으로 뛰쳐들어감 … 예전 대전 홍도동 남친집 …
그때는 삼재였지만…

지금은 어케 살아야 하나요 …

ㅇㅍ는 못뛸 거 같아요 … (ㅅㅅ랑 삽입 절대 못할 듯)
하퍼 나가려구 하는데 …

가야겠죠 …?

한밭수목원 가서 명상하고 싶어요 …

대학원와서 정신없어진 것 같아요 …

아 근데 하퍼나가도 스킨십 안될 것 같아요 …

아 그냥 원래 나갔던 사무실 에서 막콜 타고 더이상 일 하지 말걸 …

저 지금 무슨 일을 해야할 지 모르겠어요 …

유흥 일은 끝난거죠 …?
